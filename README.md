# 🍷 Wine Quality Analysis using Python

## 📌 Project Overview

This project focuses on analyzing the **Wine Quality Dataset** using Python.
The main objective is to explore the dataset and generate meaningful insights through **data visualization**.

The project includes **10+ graphs** to understand how different chemical properties affect wine quality.

---

## 📂 Dataset

* Dataset used: `winequality-red.csv`
* Contains physicochemical properties of red wine such as:

  * Alcohol
  * pH
  * Density
  * Sulphates
  * Acidity levels
  * Quality (target variable)

---

## 🎯 Objectives

* Perform **Exploratory Data Analysis (EDA)**
* Understand relationships between features
* Identify factors affecting wine quality
* Visualize data using graphs

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* Matplotlib

---

## 📈 Key Insights

* Higher alcohol content is associated with better quality wine
* Volatile acidity negatively impacts wine quality
* Most wines fall in the medium quality range (5–6)
* Strong correlations exist between several chemical properties
